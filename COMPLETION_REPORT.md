## 📞 QUICK START COMMANDS

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
# Backend will run on http://localhost:5000
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env.local
npm run dev
# Frontend will run on http://localhost:3000
```
